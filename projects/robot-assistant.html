Overview

Built a natural-language controlled robotic assistant using xAI APIs for the 2025 xAI Hackathon. The system lets you speak a command — "Can you pass me the banana?" — and watch a robotic arm find, grasp, and deliver the object autonomously. The whole pipeline runs on consumer hardware: a standard webcam, an off-the-shelf robotic arm, and a laptop.

How It Works
Perception Webcam feed processed by a real-time object detection pipeline to identify and locate common objects on a table surface.
Agent Grok acts as a Vision-Language-Action (VLA) agent. LangChain tools give it access to real-time robot state data and the detection pipeline — it reasons about what it sees and decides how to act.
Voice Voice API handles natural speech input. Commands are transcribed and passed directly to the agent.
Hardware Off-the-shelf robotic arm with a custom gripper designed in OnShape — optimized for reliable object grasping on a flat surface.
Actuation Agent outputs translate to motor commands. The arm moves to the target, grasps the object, and returns — closed-loop from speech to physical action.
What I Built
  • Custom gripper designed in OnShape — mounted on an off-the-shelf robotic arm, optimized for grasping common household objects.
  • Webcam-based object detection pipeline integrated into the agent's tool loop, providing real-time scene awareness.
  • LangChain tool integrations exposing robot joint state, gripper status, and vision output to the Grok VLA agent.
  • End-to-end voice-to-actuation pipeline demonstrated live at the hackathon.
